# 汽车站车辆运营管理系统 java891

## 项目概述

汽车站车辆运营管理系统是基于 JavaEE 的 SSM+Vue 的前后端分离项目。该项目采用 MVC 三层体系结构，结合 Spring、SpringMvc、Mybatis 与 Vue 技术，使用 Maven 进行项目管理。系统后端采用 MySQL 数据库进行数据存储，前端通过 Web 界面实现基于 B/S 架构的管理功能。整体界面简洁，操作方便。

## 技术栈

- 后端：Spring、SpringMvc、Mybatis
- 前端：Vue
- 数据库：MySQL
- 项目管理：Maven

## 功能模块

- 车辆管理
- 运营管理
- 系统管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- JDK 1.8 或更高版本
- MySQL 5.7 或更高版本
- Maven 3.6 或更高版本
- Node.js 12.0 或更高版本

## 部署步骤

1. 创建数据库，导入 MySQL 数据库脚本
2. 修改配置文件中的数据库连接信息
3. 使用 Maven 编译项目
4. 部署至 JavaEE 服务器（如 Tomcat）
5. 前端项目编译并部署至 Web 服务器
